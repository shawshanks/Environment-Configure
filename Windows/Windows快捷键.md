## Alt
Alt + Enter (或 鼠标左键）  查看所选项目的属性  
Alt+F4                      关闭当前程序  
Alt+D                   选择地址栏（资源管理器）  
Alt+Spacebar            快捷菜单  

## Ctrl
Ctrl+A                     选择所有  
Ctrl+W                  关闭当前窗口（资源管理器）  
Ctrl+E                  选择搜索框（资源管理器）  
Ctrl+Esc                开始屏幕  
Ctrl+Shift+Esc          任务管理器  

## win键
Windows 键：打开开始屏幕  
win+ .    表情 😊  
win + tab 切换应用  
win + E   打开资源管理器  

win + M: 最小化所有窗口  
Windows 键+Shift+M：在桌面恢复所有最小化窗口
